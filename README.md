# OPERATION: AI VANGUARD - COMMAND HUB

This repository houses the static frontend for the AI Vanguard mission command board. It is designed to be lightweight, heavily tactical, and strictly informational.

## DEPLOYMENT ORDERS (GitHub Pages)
1. Navigate to **Settings** in this repository.
2. Select **Pages** from the left-hand sidebar.
3. Under **Build and deployment**, set the Source to **Deploy from a branch**.
4. Select the `main` branch and the `/ (root)` folder.
5. Click **Save**.

Your Forward Operating Base will be live at `https://[your-username].github.io/[repo-name]/` within 60 seconds.

## IDEA FUNNEL (Target Acquisition)
To pitch a new micro-target, engineers must open a New Issue in this repository using the standard format. If it does not pass the 3-question filter (Data Ready? Under 15 Hrs? 2+ Hr ROI?), it will be closed immediately.
